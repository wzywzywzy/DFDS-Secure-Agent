# Blake's slot — five attack-family slide images

Drop-in replacements for the devil-on-envelope image that currently
sits on the right side of Blake's "Poisonous emails" slide. As Blake
speaks each bullet, click to swap the image so the audience sees a
different villain pop out of the envelope.

## All 5 categories are covered by the original 9

| # | Category | Image | Original-9 examples |
|---|---|---|---|
| 1 | Prompt injection      | `01_prompt_injection.png`     | poison_01, 02, 08 |
| 2 | Authority pressure    | `02_authority_pressure.png`   | poison_07 |
| 3 | Unauthorized action   | `03_unauthorized_action.png`  | poison_03, 04 |
| 4 | Out-of-scope · exfil  | `04_out_of_scope.png`         | poison_05, 06 |
| 5 | Identity fraud        | `05_identity_fraud.png`       | poison_09 |

## How to use in PowerPoint / Keynote

1. Stack all 5 images in the same spot on the slide where the
   devil emoji currently sits.
2. Set animations on each image so they appear in order, **on
   click**, and the previous one disappears.
3. As Blake speaks each bullet on the left, click — the image on
   the right swaps. Five bullets, five images, five clicks.

In Keynote: Build In = Appear (with click), Build Out = Disappear
(with next click).
In PowerPoint: Animations → Appear → Start: On Click; for each
preceding image add Disappear at the same trigger.

## Style note

Generated by `render.py` in this folder using the macOS Apple Color
Emoji glyph as the central icon, on a soft tinted gradient backed by
a subtle envelope outline. The envelope motif is intentional — keeps
visual continuity with the existing slide's purple-envelope-with-
devil image.

If you want a premium 3D-rendered look that matches the existing
devil-on-envelope exactly, regenerate each one in Midjourney / DALL-E
with prompts of the form:

```
3D rendered envelope with a small soft <emoji name> emerging from
the opening, soft purple gradient background, isometric view,
playful cartoon style, no text, transparent background, 1024x1024
```

Replace `<emoji name>` per image:
- 01: "robot face with a hijack sticky note on its forehead"
- 02: "fake CEO crown with a red urgency clock"
- 03: "Trojan horse"
- 04: "leaking water droplet over a database"
- 05: "theatre comedy/tragedy masks"

## Regenerating

```bash
python docs/slide_images/render.py
```

Edits the 5 images in place. Tweak `CATEGORIES` in `render.py` to
change emoji choices, captions, or tint.
